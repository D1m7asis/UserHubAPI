UserHub - API управления пользователями

Добро пожаловать в сервис управления пользователями. Доступные методы API:

POST /users

Создать нового пользователя. Требуются имя, фамилия и пароль.

GET /users

Получить список всех пользователей.

GET /users/{id}

Получить конкретного пользователя по ID.
